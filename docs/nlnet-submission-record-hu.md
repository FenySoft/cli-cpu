## Beadott synopsis

Az NLnet-hez beadott teljes szöveg egy **tömörített synopsis** (kb. 3 oldal, nem a repó ~10-oldalas kibővített draft-ja).

- A **kibővített draft** ([`nlnet-application-draft-hu.md`](nlnet-application-draft-hu.md)) szolgált forrásanyagként.
- A **beadott form** egy rövidebb synopsis verziót transzkribált ugyanabból a tartalomból, az NLnet beadási forma struktúrájához igazítva (synopsis / experience / usage / comparison / challenges / ecosystem).

## Ismert eltérések a hiteles drafttól

Négy tartalmi szintű hiba keletkezett a beadási transzkripció során — lásd [`nlnet-corrections-hu.md`](nlnet-corrections-hu.md) §11-14 a részletekért:

1. A cím kihagyja a „Processing Unit (CFPU)" kibővítést
2. „OPI" jelenik meg ott, ahol „QSPI" volt szándékolva (§13)
3. A QSPI latency „6-10 ciklus"-ként szerepel a helyes „10-50 ciklus" helyett (§13)
4. Magyar ékezetek elvesztek tulajdonnevekben („Adougyi Ellenorzo Egyseg", „MAV") — §14

Ezek nem érintik a pályázat lényegét, de tisztázandók, ha az NLnet bírálók kérdeznek.
